School Projects

A selection of projects developed during my Diploma in Game Development & Technology at NYP. Each project reflects skills built across programming, systems design, and technical art.

Computer Graphics Simulation Project

A 3D carnival-themed game built with OpenGL, combining rendering and physics to create interactive minigames. Covered 3D model rendering, object-oriented architecture, and real-time physics programming as a group project.

OpenGL C++ OOP 3D Rendering Physics

2D Game Project - SG60

A Unity 2D group project themed around Singapore's SG60 celebration, showcasing local cultural cuisines through minigames. Features a main lobby with an upgrade shop and unlockable food facts upon completing each minigame.

Unity C# 2D Game Design Audio Design UI Design

3D Game Project - FPS Boss Fight

A solo FPS boss fight demo built in Unity 3D. Implemented FPS player movement and weapon mechanics, NavMesh AI pathfinding, a finite state machine for enemy behaviour, 3D map design, and custom HLSL shaders using URP Shader Graph.

Unity C# NavMesh AI FSM HLSL / URP 3D Map Design

3D Stealth Level Design

To test my own understanding of stealth level design and 3D level design in general, I used unreal engine 5 to design a map that utilizes lighting and spaces to guide and lead players through a stealth map, with only base knowledge about unreal engine 5 as well as 3D game level design concepts.

Unreal Engine 5 Blueprints 3D Game Level Design

3D Map Design Analysis

Additionally to my 3D stealth game level design, I also made a video to analyse and show the clever level design choices of the popular Roblox Game, Pressure. The video was optional as a document is enough to suffice, but I wanted to also showcase my video editing skills alongside the assignment.

3D Game Level Design Video editing Alight Motion

Final Year Project - Multiplayer Roguelike

A 3D co-op multiplayer roguelike where players survive infinitely generated maps against AI monsters. All progress is lost on a full party wipe. Features networked multiplayer via Unity Netcode, NavMesh AI enemies, and art assets from Game Visual Effects teammates.

Unity 3D Multiplayer Netcode C# NavMesh AI Procedural Generation
